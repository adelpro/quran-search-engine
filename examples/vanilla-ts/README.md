# Vanilla TypeScript Example

A simple vanilla TypeScript example demonstrating the Quran Search Engine library.

## Features

- Search through Quran verses
- Multiple search modes: exact, lemma, root, and fuzzy matching
- Real-time search with debouncing
- Highlighted search results

## Getting Started

1. Install dependencies:

   ```bash
   pnpm install
   ```

2. Start the development server:

   ```bash
   pnpm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`

## Usage

- Enter Arabic text in the search box
- Toggle search options (Lemma, Root, Fuzzy) as needed
- View highlighted results with match scores and types
